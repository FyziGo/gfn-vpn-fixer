# Nvidia Geforce Now VPN FIXER

[English](README.md) | **Русский** | [Español](README.es.md)

Утилита для Windows, которая автоматически отключает указанные сетевые адаптеры и VPN-сервисы во время работы GeForce NOW, а затем включает их обратно при закрытии GFN. Это предотвращает влияние VPN/туннельного ПО (Tailscale, ZeroTier, WireGuard и др.) на стриминг GeForce NOW.

## Возможности

- **Автоматическое управление сетью** — отслеживает процесс `GeForceNOW.exe` и переключает адаптеры/VPN на лету
- **Ограничение скорости** — ограничивает пропускную способность для GFN через Windows QoS для стабилизации битрейта
- **Системный трей** — работает тихо в фоне с иконкой в трее (зелёная = GFN активен, серая = ожидание)
- **GUI-настройки** — окно настроек на Fyne для выбора адаптеров, VPN-сервисов и лимита скорости
- **Автозапуск** — интеграция с Планировщиком задач Windows с правами администратора (без UAC-запроса при входе)
- **Без мерцания консоли** — все команды PowerShell выполняются в скрытых окнах

## Требования

- Windows 10/11 (Pro или выше рекомендуется для ограничения скорости через QoS)
- Права администратора (UAC-запрос при первом запуске)

## Использование

```powershell
# Открыть GUI настроек (первый запуск или для изменения параметров)
.\gfn-vpn-fixer.exe --setup

# Фоновый режим (обычная работа, иконка в системном трее)
.\gfn-vpn-fixer.exe
```

## Сборка из исходников

**Требования:**
- Go 1.22+

## Конфигурация

Настройки хранятся в `config.json` рядом с исполняемым файлом:

```json
{
    "blacklist": ["Имя адаптера 1", "Имя адаптера 2"],
    "gfn_path": "C:\\...\\GeForceNOW.exe",
    "launch_gfn": false,
    "vpn_services": ["Tailscale", "ZeroTierOneService"],
    "bandwidth_limit_mbps": 100
}
```

| Поле | Описание |
|------|----------|
| `blacklist` | Имена сетевых адаптеров для отключения |
| `gfn_path` | Путь к GeForceNOW.exe |
| `launch_gfn` | Автозапуск GFN вместе с программой |
| `vpn_services` | Имена Windows-сервисов для остановки |
| `bandwidth_limit_mbps` | Ограничение скорости в Мбит/с (0 = без ограничения) |

## Принцип работы

1. При запуске проверяет права администратора (перезапуск с UAC при необходимости)
2. Если указан `--setup` или отсутствует конфиг → открывается GUI
3. Иначе → фоновый режим с иконкой в системном трее
4. Отслеживает `GeForceNOW.exe` каждые 2 секунды через Windows API (`CreateToolhelp32Snapshot`)
5. При обнаружении GFN:
   - Останавливает настроенные VPN-сервисы
   - Отключает адаптеры из чёрного списка
   - Применяет ограничение скорости через QoS (если настроено)
6. При закрытии GFN → возвращает все изменения обратно
